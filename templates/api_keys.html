API密钥使用说明

API密钥可以通过以下两种方式之一提供：

  1. 在HTTP请求头中添加 X-API-Key 头
  2. 在URL查询参数中添加 api_key 参数

示例：

curl -H "X-API-Key: 您的API密钥" {{ request.host_url }}api/random/5

{{ request.host_url }}api/random/5?api_key=您的API密钥
